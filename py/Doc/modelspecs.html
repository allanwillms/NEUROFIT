Model Specifications Panel

There are three values that need to be filled in to specify the model:
  1. The number of inactivating channel groups (nh )
  2. The number of non-inactivating channel groups (nnonh )
  3. The number of independent activation gates ( p )
1. The number of inactivating channel groups must be an integer between zero and three. As nh increases, the problem of fitting the data becomes ill-posed and satisfactory results are unlikely.

2. The number of non-inactivating channel groups must be either zero or one, but must be one if nh is zero.  For example, to fit data from a channel which has no inactivation, set nh to zero and nnonh to one. For data which has two time constants of inactivation and a residual non-inactivating component, set nh to two and nnonh to one.

3. The number of independent activation gates can be any positive integer. Although this number has historically been often set to three or four, unless your data shows a distinct delay in the rise of the current at the start of the voltage step, p should be set to one.
